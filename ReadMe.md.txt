# Messenger Projekt

Dieses Projekt ist ein Messenger, der es Benutzern ermöglicht, miteinander zu kommunizieren und ihre Konversationsverläufe zu speichern.

## Projektüberblick

Das Ziel dieses Projekts ist es, einen Messenger mit verschiedenen Funktionen zu entwickeln, um die Kommunikation zwischen Benutzern zu erleichtern. Das Projekt umfasst die Implementierung von Vererbung, Overriding von Methoden, File IO, Multithreading, Networking und eine benutzerfreundliche GUI.

## Anforderungen

### Must-haves

- Benutzer können sich mit Credentials registrieren und einloggen.
- Auswahl und Anschreiben von anderen Benutzern.
- Speicherung von Konversationsverläufen, die nach erneutem Einloggen wieder geladen werden können.

### Should-haves

- Suche nach Benutzern in der Liste.
- Durchsuchen von Nachrichten innerhalb eines Chats.
- Möglichkeit für Benutzer, Profilbilder hochzuladen und zu ändern.

### Nice-to-haves

- Erstellung von Channels/Rooms mit mehr als 2 Benutzern.
- Editierbare Namen für Channels/Rooms.
- Markierung von Nachrichten.

### Overkill

- Anzeige von Benutzerstatus.
- Weiterleitung von Nachrichten.
- Nachträgliches Bearbeiten von Nachrichten.
